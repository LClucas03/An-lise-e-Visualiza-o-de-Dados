# Análise e Visualização de Dados

Este projeto realiza a análise e visualização de dados financeiros, incluindo preços de abertura, fechamento e volume de negociação. O projeto é implementado em Python e utiliza as bibliotecas `pandas` e `matplotlib` para processamento e visualização dos dados.

## Estrutura do Projeto

- `src/plot_data.py`: Contém o código para leitura dos dados, análise e geração dos gráficos.
- `README.md`: Este arquivo.
- `.gitignore`: Arquivo para ignorar arquivos e pastas específicos do Git.
